# MrMarkuz apps index for NS8

This is the official NS8-MrMarkuz repository - the index of apps.

Metadata are built every 4 hours at 00:25, 06:25 ,12:25, 18:25 UTC and on each commit to the `main` branch.

To add this repository, see the [manual
page](https://docs.nethserver.org/projects/ns8/en/latest/software_center.html#software-repositories)
and set the following URL:

    https://repo.mrmarkuz.com/ns8/updates/
